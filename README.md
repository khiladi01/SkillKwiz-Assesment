# SkillKwiz

SkillKwiz is a frontend technical assessment project built using **Next.js (App Router)** and **Tailwind CSS**. It showcases a modern, responsive UI with reusable components.

## Installation
Clone the repository:

bash
git clone https://github.com/khiladi01/SkillKwiz-Assesment.git

## Live
Deploy on Vercel

## Features
- Fully responsive frontend using Tailwind CSS
- Modular and reusable UI components
- Clean folder structure with Next.js App Router
- Dynamic pages with navigation
- Easy to extend for further functionality

## Tech Stack
- **Next.js** (App Router) – for server-side rendering and routing
- **React Components** – modular UI
- **Tailwind CSS** – utility-first styling
- **Vercel** – deployment platform

## Project Structure

SkillKwiz/
├─ app/ # Main Next.js App Router folder
│ ├─ components/ # Reusable UI components
│ ├─ pages/ # Dynamic pages (if any)
│ ├─ styles/ # Tailwind CSS custom styles
│ └─ layout.js # Main layout file
